Assistente Virtual: Perséfone

Inteligência Artificial

Neste projeto eu construí um programa de inteligência artificial para me ajudar com 3 tarefas recorrentes do meu dia a dia. O projeto consiste em uma assistente virtual que através de comandos de voz podem: abrir programas, realizar pesquisas no google e realizar pesquisas no youtube. Como são atividades básicas eu utilizei apenas as bibliotecas do PyAutoGui, PyperClip e PyWhatKit.

Antes de rodar os programas em uma máquina é preciso utilizar o comando "pip install -r requirements.txt" para ter acesso às bibliotecas necessárias para o funcionamento do programa.

Ao utilizar o programa as funções de realizar pesquisas no google e realizar pesquisas no youtube vão rodar independente de qual seja a máquina pois são genéricas pórem como esse projeto foi feito pensando exclusivamente com o propósito de rodar na minha máquina, os endereços de atalhos dos aplicativos a serem abertos são referentes aos do meu computador. Então caso um usuário baixe o programa, é possível adaptar esses endereços mas se optar por não alterar o programa irá reconhecer que não há atalho salvo e irá abrir o menu iniciar para pesquisar pelo aplicativo em questão e abrir ele.
